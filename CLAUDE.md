# Project Guidelines

## Commit Message Format

```
type(#issuenum) : content
```

**Examples:**
- `feat(#12) : 점수 시스템 구현`
- `fix(#7) : 과일 충돌 버그 수정`
- `refactor(#4) : MergeResolver 싱글톤 제거`
- `asset(#3) : 과일 에셋 임시 스프라이트 적용`

**Types:** `feat`, `fix`, `refactor`, `asset`, `docs`, `chore`, `style`, `test`
